DOM 'Document Object Model' (Modelo de Objetos para Documentos) Um conjunto de objetos dentro do seu navegador que te dará acesso aos componentes internos da web Se faz presente quando o JavaScript está rodando na web. Seleção: Por Marca = document.getElementsByTagName() Por ID = document.getElementById() Por Nome = document.getElementsByName() Por Classe = document.getElementsByClassName(.) Por seletor = document.querySelector() document.querySelectorAll()
